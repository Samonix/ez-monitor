# Бачення продукту

EZ MONITOR - вебкабінет для мережі АЗС, який об'єднує PTS-2 telemetry, операційний live-monitoring, карткову систему, alerts, звіти, інтеграції та майбутній AI-доступ до read-only analytics.

## MVP value

- Бачити стан мережі АЗС в одному кабінеті.
- Приймати automatic upload від PTS-2 без static IP на АЗС.
- Аналізувати транзакції, резервуари, alerts і картки.
- Мати фундамент для notifications, chatbots і external API.
