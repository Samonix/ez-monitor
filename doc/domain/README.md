# Domain

Словник і бізнес-правила EZ MONITOR.

## Базові поняття

- Organization - мережа або клієнт.
- Station - конкретна АЗС.
- Controller - PTS-2 пристрій на АЗС.
- ПРК / Pump - паливороздавальна колонка.
- Tank - резервуар.
- Card/Tag - картка або RFID/tag для авторизації відпуску.
- Ledger - фінансовий/об'ємний рух по картці.
- Alert - подія або аварійний стан.
